常用命令行大全
===

- 创建文件夹               
    `mkdir test`

- 进入文件夹           
    `cd test`

- 显示路径     
    在需要显示的文件夹中执行 `pwd`
    
- 创建空文件	   
    `touch fileName`
	
- 查看文本内容         
    `cat xxx.txt`
	
- 删除文件       
    `rm xxx.txt`              
	`rm -rf file`// -fr表示递归和强制，一定要小心使用 rm -fr / 那你的电脑就over了
	
- 拷贝           
    `cp -R 源文件  目标文件`// -R 表示对目录进行递归操作
	
- 移动               
    `mv filePath toPath`
	
- 查看文件内容            
    `cat fileName`
	
- 清楚命令行内容          
    `clear`
	
- 显示日期          
    `date`
	
- 显示文件               
    `ls`                
    `ls -a`// 显示隐藏文件	
	`ls -l`列出详细信息
	
- 关机                       
    `sudo shutdown -h now`// -h 是关闭电源 now立即关机            
    `sudo shutdown -r now`//重启	    
	`sudo shutdown -h -time 60`// 表示60分钟后关机，注意单位是分钟
